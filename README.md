<h1>GTA5 Solo Session</h1>
<h2>How to use?</h2>
<ol type="I">
  <li>For the program to work, please intall <a href="https://docs.microsoft.com/en-us/sysinternals/downloads/pstools" target="_blank" rel="external">PsTools</a>.</li>

</ol>
